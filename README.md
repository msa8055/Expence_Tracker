# Expence_Tracker

A modern and responsive Expense Tracking Web App built using HTML, CSS, and JavaScript.
It helps you track income and expenses, view totals, filter transactions, and analyze spending using interactive charts powered by Chart.js.
All data is saved locally using localStorage, so nothing is lost when you refresh the page.
